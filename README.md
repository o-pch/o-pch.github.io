# o-pch.github.io

このリポジトリは GitHub Pages でホストするためのシンプルな静的サイトのテンプレートを提供します。

主なファイル:

- `index.html` - トップページ
- `404.html` - 404 ページ
- `assets/css/style.css` - スタイル
- `assets/js/main.js` - 簡単なインタラクション（年表示、モバイルメニュー）
- `assets/img/favicon.svg` - サンプルファビコン
- `.nojekyll` - Jekyll を無効化（必要に応じて）

使い方:

1. リポジトリ設定 -> Pages で `main` ブランチを公開に設定します。
2. もしサブフォルダで公開する場合は、リンクパス（先頭のスラッシュ）を調整してください。
3. 必要に応じて `index.html` を編集してコンテンツを追加してください。

カスタマイズ案:

- ナビゲーションを自動生成
- ブログ機能（静的ジェネレータ導入）
- CI/CD を追加して自動ビルド

ライセンス: 任意。自由に編集して使用してください。
